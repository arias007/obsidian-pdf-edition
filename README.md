# OB PDF Ink Xodo

给 Obsidian PDF 阅读器和笔记内嵌 PDF 加一个笔头按钮，用于手写批注和高亮，效果目标接近 Xodo 的基础墨迹标注。

## 功能

- 在 PDF 顶部菜单增加笔头按钮；笔记内嵌/引用 PDF 也会显示标注按钮。
- 点击后在 PDF 页面上叠加手写画布。
- 支持钢笔、高亮笔、选择、橡皮、撤销、清空未保存笔迹。
- 主工具条只保留一个调色板按钮，颜色、粗细、钢笔透明度、高亮透明度都在调色板里调整。
- 触摸屏上单指用于标注，双指用于滑动 PDF。
- PDF 打开后会短时补扫挂载按钮；书写过程中暂停页面重扫，减少刷新感。
- 画完后自动保存到原 PDF，不再用保存按钮占工具条空间。
- 选择工具可点击标注外框内区域选中当前会话中的标注，显示框选并让选中标注半透明。
- 画完新笔迹不会自动选中新笔迹，避免刚写完就出现选框或半透明状态。
- 去掉后台 1.2 秒轮询扫描，避免 PDF/嵌入区域不停刷新。
- 增强 PDF 菜单区域高度、层级和按钮点击区域，减少被 Spaced Repetition 等界面上下挤压后点不到按钮的问题。

## 安装

1. 关闭 Obsidian 或至少先关闭目标 Vault。
2. 把 `ob-pdf-ink-xodo` 文件夹复制到：

   ```text
   E:\note\.obsidian\plugins\ob-pdf-ink-xodo
   ```

3. 打开 Obsidian。
4. 设置 → 第三方插件 → 关闭安全模式或允许第三方插件。
5. 启用 `OB PDF Ink Xodo`。

## 使用

1. 在 Obsidian 打开 PDF。
2. 点击 PDF 顶部菜单里的笔头按钮。
3. 在页面上直接写字或划高亮。
4. 画完后会自动保存到原 PDF。

## 注意

- 自动保存会修改原 PDF。第一次建议先用 PDF 副本测试。
- 当前版本重点支持自由手写和高亮墨迹，不是完整 Xodo 替代品。
- 保存后的墨迹是烧录到 PDF 页面上的线条，不是专业 PDF 注释对象。
- 当前会话中刚画的墨迹可通过外框范围选择；关闭或重载 PDF 后，已烧录到 PDF 的线条不再作为独立对象编辑。
- 如果 PDF 加密、只读、被其他程序占用，保存可能失败。
- 连续快速书写会合并延迟保存，避免每一笔都立刻重写 PDF。
